- What does it’s call do ?

The fork() system call creates a new process by duplicating the calling process.
The original process becomes the parent, and the new one is the child

- What kind of partitions do you seem in manual page ?
The fork() manual page contains standard sections such as:

NAME - name and short description
SYNOPSIS - function usage and required headers
DESCRIPTION - detailed explanation
RETURN VALUE - values returned to parent and child
ERRORS - possible error conditions
CONFORMING TO - applicable standards (e.g. POSIX)
NOTES / SEE ALSO - additional information and related calls
